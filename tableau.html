DIRECTOR GENDER DIVERSITY DASHBOARD

This dashboard analyzes the gender diversity in films for the top female directors. The filters or the interactive graphs can be used to investigate the gender diversity for a specific director, actor, film or year.

FILM LANGUAGE DASHBOARD

This dashboard examines the most common languages for films directed by the top female directors. The graphs can be analyzed to see how many films are available for a particular language or how the number of films produced for a language has changed throughout time. The filters or search box can be used to examine a particular director, film, or language.
